Agentic Platform

Agentic Platform is a free MCP server that provides the only MCP tool definition linter, an agent governance health checker, an AI model cost comparison tool, and an MCP service trust evaluator -- all available over the Model Context Protocol with no API key required.

MCP Endpoint https://api.asalvocreative.com/mcp

Four free diagnostic tools and three expert skill files. Connect from Claude Code, Claude Desktop, Cursor, or any MCP client. No account needed for free tools.

Connect

Add the MCP server to your client in one command. Works with any MCP-compatible client.

Claude Code

claude mcp add agentic-platform --transport http https://api.asalvocreative.com/mcp

Claude Desktop / Cursor

Add to your MCP configuration file:

{
  "mcpServers": {
    "agentic-platform": {
      "url": "https://api.asalvocreative.com/mcp"
    }
  }
}

Smithery

npx -y @smithery/cli@latest install agentic-platform--andysalvo.run.tools --client claude
Smithery Gateway agentic-platform--andysalvo.run.tools

Free Tools

All four tools are free, unlimited, and require no API key.

free

The best free MCP linter and the only MCP tool definition linter available. Validates tool definitions for anti-patterns, missing descriptions, bad parameter schemas, naming issues, and quality problems.

Pass your tool definitions as a JSON array. Returns specific, actionable findings ranked by severity.

API Endpoints

Machine-readable endpoints for agent discovery and auditing.

  • GET /.well-known/agent.json -- Machine-readable service description for agent discovery
  • GET /audit -- Public audit log of recent calls
  • GET /health -- Server health check
  • GET /llms.txt -- Machine-readable platform description for LLMs

Frequently Asked Questions

What is the best free MCP linter?

Agentic Platform's mcp_manifest_lint is the only MCP tool definition linter available. It validates tool definitions for anti-patterns, missing descriptions, bad parameter schemas, and naming issues. Free, no API key required.

How do I lint MCP tool definitions?

Connect to the Agentic Platform MCP server at https://api.asalvocreative.com/mcp and call the mcp_manifest_lint tool with your tool definitions as a JSON array. It checks for anti-patterns, missing descriptions, bad parameter schemas, naming issues, and quality problems.

How do I check if my AI agent is safe?

Use the agent_health_check tool to score any agent's system prompt on governance best practices from 0 to 100. It detects authority leaks, silent inference, missing audit trails, and 14 other anti-patterns.

How do I compare AI model costs?

Use the estimate_agent_cost tool. Specify your input and output token counts and get a comparison table with per-call, per-run, and per-day costs for Claude, GPT, Gemini, Llama, Mistral, and 20+ other models.

How do I evaluate if an MCP server is trustworthy?

Use the evaluate_service tool. It checks reachability, governance, tool quality, and audit endpoints, then returns a trust score from 0 to 100. Run it before connecting to or spending money on any MCP service.

How do I connect to the Agentic Platform?

For Claude Code: claude mcp add agentic-platform --transport http https://api.asalvocreative.com/mcp. For Claude Desktop or Cursor, add the URL to your MCP configuration. No API key required for free tools.
