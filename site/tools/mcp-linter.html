MCP Manifest Linter

The only MCP tool definition linter

MCP Endpoint https://api.asalvocreative.com/mcp

The mcp_manifest_lint tool validates your MCP tool definitions against quality standards. It is the only linter purpose-built for MCP tool definitions. Free to use, no API key required.

What It Checks

  • Missing or inadequate tool descriptions
  • Bad parameter schemas (missing types, descriptions, constraints)
  • Naming anti-patterns (inconsistent casing, ambiguous names)
  • Missing annotations (readOnlyHint, destructiveHint, etc.)
  • Overly broad or vague parameter definitions
  • Tool definitions that would confuse an agent during tool selection

Usage

Connect to the Agentic Platform MCP server and call mcp_manifest_lint with your tool definitions as a JSON array.

Example Input

mcp_manifest_lint({
  tools_json: '[{"name": "do_stuff", "description": "", "inputSchema": {"type": "object"}}]'
})

Example Output

MCP Manifest Lint Report
========================

Tool: do_stuff
  FAIL  Missing description
  FAIL  Vague tool name -- agents cannot infer purpose
  WARN  No inputSchema properties defined
  WARN  Missing annotations (readOnlyHint, destructiveHint)

1 tool checked. 2 errors, 2 warnings.

About the Agentic Platform

The Agentic Platform is an MCP server that provides free agent diagnostics and expert skill files. In addition to the MCP linter, it offers an agent health check, cost estimator, and service trust evaluator -- all free, no API key required.
