Service Trust Evaluator

Evaluate any MCP server before spending money on it

MCP Endpoint https://api.asalvocreative.com/mcp

The evaluate_service tool connects to any MCP server and evaluates it for trustworthiness. It checks reachability, governance declarations, tool definition quality, and audit endpoints. Returns a trust score from 0 to 100 with a clear recommendation. Free to use, no API key required.

What It Checks

  • Server reachability -- can the MCP endpoint be reached and does it respond correctly
  • Governance declarations -- does the server publish governance invariants
  • Tool definition quality -- are tool descriptions, schemas, and annotations well-formed
  • Audit endpoints -- does the server expose an audit log or transparency endpoint
  • Agent.json -- does the server publish a /.well-known/agent.json file
  • HTTPS -- is the connection encrypted

Trust Recommendations

  • PROCEED (75-100) -- Server meets governance standards
  • PROCEED WITH CAUTION (50-74) -- Some governance gaps, review before transacting
  • HIGH RISK (25-49) -- Significant governance concerns
  • DO NOT TRANSACT (0-24) -- Server does not meet minimum standards

Usage

Example Input

evaluate_service({
  server_url: "https://example.com/mcp",
  task_context: "I need to use this server for code generation"
})

About the Agentic Platform

The Agentic Platform is an MCP server that provides free agent diagnostics and expert skill files. In addition to the service evaluator, it offers an MCP linter, agent health check, and cost estimator -- all free, no API key required.
