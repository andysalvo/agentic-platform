Agent Health Check

Governance scoring for agent system prompts, 0 to 100

MCP Endpoint https://api.asalvocreative.com/mcp

The agent_health_check tool scores any agent's system prompt on governance best practices. It returns a detailed diagnostic report with specific issues found, severity ratings, and actionable fixes. Free to use, no API key required.

What It Checks

  • Authority leaks -- places where an agent claims or assumes authority it was not given
  • Silent inference -- decisions made without explicit reasoning or user awareness
  • Missing audit trails -- no mechanism for logging or reviewing agent decisions
  • Scope creep -- agent instructions that expand beyond stated purpose
  • Missing error handling -- no guidance for what to do when things go wrong
  • Vague delegation -- unclear boundaries on what the agent can and cannot do
  • 14 additional governance anti-patterns

Usage

Connect to the Agentic Platform MCP server and call agent_health_check with your agent's system prompt.

Example Input

agent_health_check({
  system_prompt: "You are a helpful assistant. Do whatever the user asks. Use any tools available to complete tasks efficiently."
})

Example Output

Agent Health Check Report
=========================
Score: 28/100

  3. Include error handling instructions

About the Agentic Platform

The Agentic Platform is an MCP server that provides free agent diagnostics and expert skill files. In addition to the health check, it offers an MCP linter, cost estimator, and service trust evaluator -- all free, no API key required.
