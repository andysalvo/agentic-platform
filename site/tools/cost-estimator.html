Agent Cost Estimator

Compare AI model costs for any agent task

MCP Endpoint https://api.asalvocreative.com/mcp

The estimate_agent_cost tool compares the cost of running an agent task across all major AI models. It returns a comparison table with per-call, per-run, and per-day costs, plus optimization tips based on your task description. Free to use, no API key required.

Models Covered

  • Anthropic: Claude Opus, Sonnet, Haiku
  • OpenAI: GPT-4o, GPT-4o mini, o1, o3
  • Google: Gemini 2.5 Pro, Gemini 2.5 Flash
  • Meta: Llama 4 Maverick, Llama 4 Scout
  • Mistral: Large, Small, Codestral
  • Additional models as they are released

Parameters

  • model -- Model name to highlight in the comparison (optional)
  • input_tokens -- Estimated input tokens per API call (default: 1000)
  • output_tokens -- Estimated output tokens per API call (default: 500)
  • num_calls -- Number of API calls per task run (default: 1)
  • task_description -- What the agent does, for optimization tips (optional)

Usage

Example Input

estimate_agent_cost({
  model: "claude-sonnet-4",
  input_tokens: 2000,
  output_tokens: 1000,
  num_calls: 5,
  task_description: "Code review agent that reads PRs and suggests fixes"
})

Optimization tips:
  - Consider Haiku for routine reviews, Sonnet for complex ones
  - Cache system prompt to reduce input token costs
  - Batch small PRs to reduce per-call overhead

About the Agentic Platform

The Agentic Platform is an MCP server that provides free agent diagnostics and expert skill files. In addition to the cost estimator, it offers an MCP linter, agent health check, and service trust evaluator -- all free, no API key required.
